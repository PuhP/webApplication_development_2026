# Система автоматизации учебного взаимодействия и управления динамическими очередями

Веб-приложение, предназначенное для оптимизации учебного процесса в высших учебных заведениях. Проект объединяет в себе функционал поиска партнеров по обучению (Study-Buddy) и интеллектуальную систему управления очередями для сдачи лабораторных и курсовых работ.

## Основные функции

- **Умные очереди:** Алгоритм автоматического формирования очереди на основе посещаемости, количества сданных работ и времени записи.
- **Поиск напарников (Study-Buddy):** Система подбора партнеров для совместных проектов на основе профиля компетенций (стека технологий и навыков).
- **Ролевая модель:** Разграничение прав доступа для Студентов, Старост и Преподавателей.
- **Файловое хранилище:** Возможность загрузки выполненных работ студентами и их централизованная выгрузка преподавателем.
- **Интерактивный дашборд:** Отображение текущего статуса очереди и статистики успеваемости группы в реальном времени.

## Технологический стек

- **Backend:** Python 3.x, Flask (фреймворк)
- **Database:** PostgreSQL (СУБД), SQLAlchemy (ORM)
- **Frontend:** Jinja2 (шаблонизатор), HTML5, CSS3 (адаптивная верстка)
- **Deployment:** Git, Docker (планируется)
